\chapter{Chip 4017}

El 4017 es un contador de décadas.

Lo usaremos para secuenciar.

Tiene 16 patitas, las cuales son:

\begin{itemize}
    \item 01: Q_{5}
    \item 02: Q_{1}
    \item 03: Q_{0}
    \item 04: Q_{2}
    \item 05: Q_{6}
    \item 06: Q_{7}
    \item 07: Q_{3}
    \item 08: GND
    \item 09: Q_{8}
    \item 10: Q_{4}
    \item 11: Q_{9}
    \item 12: Carry
    \item 13: Enable
    \item 14: Clock
    \item 15: Reset
    \item 16: V_{CC}
\end{itemize}
