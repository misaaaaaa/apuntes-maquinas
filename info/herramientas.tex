\chapter{Herramientas}

\section{LaTeX}

Este libro está programado en \gls{latex}.

Incluye las subherramientas Tikz y Circuitikz, que permiten crear gráficos y diagramas de circuitos electrónicos.

\section{Git}

Somos gente entusiasta por el control de versiones, por lo que este libro está hecho usando \gls{git}, una herramienta de control de versiones que permite tener no solamente una versión actual del libro, sino que también una bitácora de cada cambio.

\section{Kicad}

Kicad es un software gratuito y de fuente abierto para diseño y fabricación de circuitos electrónicos.

\section{C++}

C++ es un lenguaje de programación, que usaremos para programar microcontroladores en este libro.
